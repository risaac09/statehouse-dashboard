Recommendation

Variation B (the dome) for the header mark, paired with Variation C (the tiled dome) for the favicon and PWA icon. Same dome shape across both, so the brand stays consistent from a 16px tab to a 192px home-screen tile. The rounded cupola survives downscaling far better than the three-column facade, whose verticals merge into a smudge at 16px. The Capitol-front mark (A) is the current app icon and stays a valid fallback if you want the literal facade read.
